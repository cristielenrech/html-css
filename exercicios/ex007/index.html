Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica) - não recomendado.

Nesta frase temos um termo em destaque usando a tag STRONG (semântica) - Recomendada

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (Não semântica) - Não Recomendada

Nesta frase temos um termo em ênfase usando a tag EM (semântica) - Recomendada

Texto marcado

Podemos criar um texto marcado com a tag MARK

Texto grande e pequeno

Estamos criando um texto grande com a tag BIG (obsoleta) e um texto pequeno com a tag SMALL (em uso)

Texto deletado (riscado)

Podemos criar um texto excluído para indicar que ele deve ser lido, mas não considerado, com a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag INS

Sublinhado

Existe também o sublinhado com a tag U (não semântica - não recomendada)

Texto sobrescrito

Para inserir coisas do tipo x20+3 - tag SUP

Texto subscrito

Para inserir coisas do tipo H2O - tag SUB
